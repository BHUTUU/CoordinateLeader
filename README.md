# CoordinateLeader

A simple AutoCAD .NET plugin to create **Leaders** and **MultiLeaders** with automatically generated coordinate labels.

No manual typing. Just pick a point and place the leader.

## Features

* Create coordinate **Leaders** (`IDL`)
* Create coordinate **MultiLeaders** (`IDML`)
* Automatically displays X and Y coordinates
* Fast, lightweight, and easy to use
* Built using the AutoCAD .NET API

## Usage

1. Load the DLL using `NETLOAD`.
2. Run:

   * `IDL` → Create a Leader
   * `IDML` → Create a MultiLeader
3. Pick the coordinate point.
4. Pick the text location.

That's it!

## Example

```text
X = 1234.567
Y = 987.654
```

## Why CoordinateLeader?

I created this plugin to speed up coordinate annotation in civil, survey, utility, and highway drawings. It removes repetitive typing and lets you annotate coordinates in just a few clicks.
